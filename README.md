# Global Unicorn Funding Analysis

## Project Overview
Analysis of unicorn companies worldwide to identify top startup ecosystems and industries attracting the highest funding.

## Research Question
Which countries have the strongest unicorn funding environments, and which industries are most funded in those countries?

## Key Findings
- **Top 5 Countries**: United States, China, India, United Kingdom, Germany
- **Most Funded Industries**: E-commerce, Fintech, AI, Health Tech
- **Insight**: Consumer-focused and software-driven sectors dominate global funding

## Tools & Technologies
- **Database**: MySQL (phpMyAdmin, XAMPP)
- **Data Cleaning**: SQL (CASE, REPLACE functions)
- **Visualization**: Excel Pivot Tables & Charts
- **Languages**: SQL

## Dataset
- Source: Kaggle
- Records: 1,037 unicorn companies
- Countries: 46
- Time Period: 2007-2022

## SQL Queries
See `sql_queries/` folder for:
1. Data cleaning (text to numeric conversion)
2. Top 5 countries analysis
3. Industry breakdown within top countries

## Visualizations
- Pivot chart: Top Unicorn Industries by Funding (see `visualizations/`)

## How to Reproduce
1. Import `unicorn_companies.csv` into MySQL
2. Run queries in `sql_queries/` folder in order
3. Export results to Excel and create pivot table
4. See `reports/` for full analysis
